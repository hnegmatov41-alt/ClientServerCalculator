# Клиент-серверный калькулятор

## Описание проекта
Клиент-серверное приложение на C++ с авторизацией и базой данных PostgreSQL.

## Функциональность
- Регистрация новых пользователей
- Авторизация (вход в систему)
- Вычисление арифметических выражений (+, -, *, /)
- Хранение истории операций в базе данных
- Просмотр статистики использования
- Поддержка нескольких клиентов одновременно

## Технологии
- C++
- Winsock2 (сокеты)
- PostgreSQL
- Многопоточность

## Инструкция по запуску

1. Создай базу данных PostgreSQL с именем calculator_db
2. Скомпилируй сервер: g++ server/server.cpp -o server.exe -lws2_32 -lpq
3. Скомпилируй клиент: g++ client/client.cpp -o client.exe -lws2_32
4. Запусти server.exe
5. Запусти client.exe

## Команды
- 1 - Войти
- 2 - Регистрация
- 3 - Выход
- stats - статистика
- exit - выход из аккаунта
